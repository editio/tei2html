Chapter VI

Lists

In this section, the various types of lists are demonstrated.

First, we start with a simple, unnumbered list. List items can contain all the normal in-line elements, such as notes, corrections, etc.

  • One
  • Two
  • Three
  • Four
  • Five
  • Six1
  • Seven
  • Eight
  • Nine
  • Ten

By specifying the type of the list as ordered, it becomes a numbered list.

  1. One
  2. Two
  3. Three
  4. Four

We can split a list into two columns, by putting columns(2) in the rend attribute.

  • One
  • Two
  • Three
  • Four
  • Five
  • Six
  • Seven
  • Eight
  • Nine
  • Ten
  1. Een
  2. Twee
  1. Drie
  2. Vier
  1. Vijf
  2. Zes
  1. Zeven
  2. Acht

A list with lower case roman numerals, using the n-attribute to set the number value.:

  1. Een
  2. Twee
  3. Achtenveertig
  4. Negenenveertig

A list without bullets or numbers:

  • Een
  • Twee
  • Drie
  • Vier
